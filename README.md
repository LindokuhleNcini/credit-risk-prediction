\# Credit Risk Prediction



\## Objective

Predict whether a loan applicant is a good or bad credit risk using machine learning.



\## Tools Used

\- Python

\- Pandas

\- Scikit-learn

\- Matplotlib

\- Seaborn



\## Model

\- Logistic Regression

\- Accuracy: 96.5%



\## Features

\- Data preprocessing (missing values, encoding, scaling)

\- Model training and evaluation

\- Confusion matrix visualization

\- Feature importance analysis



\## Results

The model achieved 96.5% accuracy with strong performance in predicting both good and bad credit risks.



\## Files

\- credit\_risk.py → main script

\- german\_credit\_data.csv → dataset

\- confusion\_matrix.png → evaluation

\- feature\_importance.png → feature analysis

\- Credit\_Risk\_Prediction\_Report.pdf → final report
